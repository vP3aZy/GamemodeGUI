	<h2>Description:</h2>
	This Plugin add a GamemodeGUI to your Minecraft Pocketmine Server!
	
		
<div align="center">
	<h2>Command:</h2>
</div>
<br>

<p align="center">  

```yaml
commands:
  gmg:
    description: "Open the GamemodeGUI!"
    permission: "gmg.use"
    permission-message: "§4Dafür hast du keine Rechte!"
```
</p>

<div align="center">
	<h2>Permissions</h2>
</div>
<br>

<p align="center">

```yaml
permissions:
```
