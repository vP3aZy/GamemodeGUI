<div align="center">
	<h2>Description:</h2>
	This Plugin adds an GamemodeGUI to your Pocketmine-MP Server!


<div align="center">
	<h2>Command:</h2>
</div>
<br>

<p align="center">  

```yaml
commands:
  gmg:
    description: "Open the GamemodeGUI!"
    permission: "gmg.use"
    permission-message: "§4Dafür hast du keine Rechte!"
```
</p>

<div align="center">
	<h2>Permissions</h2>
</div>
<br>

<p align="center">

```yaml
permissions:
  gmg.use:
    default: OP		
```
